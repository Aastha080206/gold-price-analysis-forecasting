# Gold Price Analysis & Forecasting System 📈🥇

An interactive gold price analysis and forecasting web application built using Python, Streamlit, and SARIMAX time-series modeling.

---

## 📌 Project Objective
To analyze historical gold price data (2014–2025) and forecast future price trends using advanced time-series techniques.

---

## 🚀 Features

- Historical gold price trend analysis
- Seasonality detection
- Best month identification
- SARIMAX forecasting model
- Interactive dashboard (Streamlit)
- CSV export functionality

---

## 📊 Technologies Used

- Python
- Pandas
- NumPy
- Plotly
- Streamlit
- Statsmodels (SARIMAX)

---

## 🔍 Forecasting Model
Implemented SARIMAX (Seasonal AutoRegressive Integrated Moving Average with Exogenous Variables) for time-series prediction with seasonality analysis.
